import

Я№юёђћь ёяюёюсюь ёфхырђќ ёъ№шэјюђ џтыџхђёџ шёяюыќчютрэшх ъюьрэфћ import:

import -window root screenshot.jpg

import їрёђќ яръхђр imagemagick.

Т ёыѓїрх, хёыш трь эѓцэю яюыѓїшђќ ёъ№шэјюђ юђфхыќэюую юъэр, тћ ьюцхђх шёяюыќчютрђќ xwininfo фыџ эрѕюцфхэшџ id юъэр. Я№юёђю тћяюыэшђх эшцхя№штхфИээѓў ъюьрэфѓ ш љИыъэшђх ьћјъющ яю юъэѓ, ёъ№шэјюђ ъюђю№юую эрфю яюыѓїшђќ.

import -window `xwininfo |grep 'Window id:' |cut -d" " -f4` screenshot.jpg

Хёыш тћ шёяюыќчѓхђх twinview шыш dualhead, я№юёђю ёфхырщђх ёъ№шэјюђ фтрцфћ ш шёяюыќчѓщђх imagemagick фыџ шѕ ёютьхљхэшџ:

import -window root -display :0.0 -screen /tmp/0.png

import -window root -display :0.1 -screen /tmp/1.png

convert +append /tmp/0.png /tmp/1.png screenshot.png

rm /tmp/{0,1}.png

Scrot, фюёђѓяхэ т extra ш яючтюыџхђ яюыѓїшђќ ёъ№шэјюђ, эрѕюфџёќ т ъюэёюыш, ђръ-цх яюффх№цштрхђ эхёъюыќъю яюыхчэћѕ ётющёђт ђшяр юя№хфхыџхьюую яюыќчютрђхыхь ђрщьх№р. Хёыш эх ѓърчрээю ф№ѓуюх, юэ ёюѕ№рэџхђ ёъ№шэјюђ т фш№хъђю№шў, т ъюђю№ющ эрѕюфшыёџ яюыќчютрђхыќ, ъюуфр чряѓёђшы ъюьрэфѓ

scrot -t 20 -d 5

бюѕ№рэџхђ ёъ№шэјюђ т .PNG єрщы, т шьхэш ъюђю№юую я№шёѓђёђтѓхђ фрђр ёъ№шэјюђр. връ-цх ёюѕ№рэџхђёџ ьшэшрђў№р (20% юђ ю№шушэрыќэюую №рёјш№хэшџ) фыџ ѓфюсэюую №рчьхљхэшџ т ёхђш. бъ№шэјюђ фхырхђёџ яюёых яџђшёхъѓэфэющ чрфх№цъш.
